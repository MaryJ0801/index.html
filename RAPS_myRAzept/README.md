## Semana 04: Innovacion Centrada en el Cliente

Esta carpeta contiene un dashboard ejecutivo en HTML sobre RAPS myRAzept y el pivote de innovacion centrada en el cliente.

## Contenido

- `index.html`: dashboard interactivo tipo executive intelligence.
- KPIs sobre adopcion digital, valor para el cliente, rendimiento de innovacion y colaboracion interfuncional.
- Visualizaciones con Chart.js para explorar customer journey, fricciones, cohortes, madurez digital y riesgos.
- Segmentacion por perfiles de usuario y lectura ejecutiva de oportunidades de mejora.

## Proposito

El archivo resume visualmente como myRAzept puede transformar datos de uso, adopcion y experiencia de cliente en decisiones para mejorar el valor entregado, reducir fricciones y priorizar iniciativas digitales.
